Second Year Tutorial: Shape Based Object Recognition

The course will discuss the problem with shape-based object recognition. How can you find an object in a cluttered image based solely on its silhoutte. Some work we've done in Bristol in this area can be found here


Topics to be discussed include:
  • What is a Shape?
  • Why Shape-Based Object Recognition?
  • Shape-Based 3D Objects
  • Shape Invariances: View-based, Scale-based, Rotation-based, Affine-based
  • Real-Time Requirements - Scalability
  • MOD Detector
  • Mobile Implementation
